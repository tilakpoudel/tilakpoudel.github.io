# tilakpoudel.github.io
Site for Tilak Poudel — Engineer, Data Scientist & Educator.

## 🚀 Setup & Local Development
Simply clone the repository and open `index.html` in your browser.


## 🛠 Features
- **Modern UI**: Built with Tailwind CSS and custom animations.
- **Human Verification**: Math-based CAPTCHA for contact form security.
- **Dark Mode**: Intelligent theme switching with local storage persistence.
- **Knowledge Network**: Interactive canvas-based skills visualization.
