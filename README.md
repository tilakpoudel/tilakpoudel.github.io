Site for Tilak Poudel — Engineer, Data Scientist & Educator.

## 🚀 Setup & Local Development

This site uses **EmailJS** for the contact form. To set it up on your local machine:

1.  **Create the credentials file**:
    ```bash
    cp assets/js/cred.js.example assets/js/cred.js
    ```
2.  **Add your keys**: Open `assets/js/cred.js` and replace the placeholder values with your actual [EmailJS](https://www.emailjs.com/) credentials.
3.  **Security**: The `cred.js` file is ignored by Git to keep your credentials out of your public history.
